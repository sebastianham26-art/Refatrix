📖 재고실사 사용방법

이 화면은 뭘 하나요?
실물 재고를 직접 세어 시스템 재고와 대조합니다. 기록만으로는 재고가 바뀌지 않습니다(감사 전용). 차이를 실제 재고에 반영하려면 마지막에 디렉터 승인이 필요합니다.
① 실사 시작
＋ 새 실사 시작 → (선택) 범위 메모 입력 → 시작. 하다가 중단하면 홈의 “진행중 실사 (이어쓰기)”에서 그대로 이어서 할 수 있습니다.
② 기록 — 랙 → 품목 → 수량 순서
1 랙 위치 칸에 랙 바코드를 스캔(또는 입력) + Enter → “현재 랙”이 표시됩니다. 다음 랙으로 이동할 때마다 다시 스캔하세요.
2 품목 코드 칸에 EAN 바코드를 스캔하거나 코드를 입력 + Enter. CTR·SYD·프로모션·EAN 모두 자동 인식됩니다.
💡 다른 랙으로 이동: 제품 스캔 중 '새 랙' 바코드를 읽으면 손 안 떼고 랙 입력으로 이동합니다. 랙 입력 옆 🏷 '새 랙' 바코드 인쇄로 한 장 뽑아 통로에 두고 쓰세요.
3 품명·시스템재고 미리보기가 뜨면 실사 수량을 입력 + Enter(저장).
＋1 스캔모드 (우상단 체크박스)
꺼짐(기본): 스캔 후 수량을 직접 입력. 박스·무더기로 세서 개수를 아는 경우 (“이 랙에 37개”).
켜짐: 스캔 1회 = 즉시 +1 누적. 미리보기·수량입력 없음. 물건을 하나씩 집으며 스캔하는 경우 (같은 코드 5번 = 5개).
③ 미등록 코드
마스터에 없는 코드도 그대로 기록됩니다(대조에서 미등록으로 별도 표시). 나중에 무엇이 안 잡혔는지 확인하는 용도예요.
④ 수정 · 삭제
기록 내역은 랙별로 묶여 보입니다. 수량은 칸에서 바로 고치고, 잘못 넣은 줄은 삭제할 수 있습니다. (제출 전까지만 가능)
⑤ 임시 대조 · 완료(제출)
🔍 임시 대조는 아직 제출하지 않고 현재까지의 차이를 미리 봅니다 — 재고에 영향이 없고, ← 기록 계속하기로 돌아가 얼마든지 추가·수정할 수 있어요. 다 세면 ✔ 실사 완료(제출)로 기록을 잠그고 최종 대조로 넘어갑니다. 항목별 실사합계 vs 시스템재고가 5가지로 분류됩니다:
일치 실사 = 시스템  ·  부족 실사 < 시스템  ·  초과 실사 > 시스템
미실사 재고는 있는데 안 센 품목(놓친 랙 탐지)  ·  미등록 마스터에 없는 스캔
상단 칩을 누르면 필터됩니다. ⬇ 엑셀로 결과를 내려받을 수 있어요. (기준은 항상 실물 재고이고, 가용재고는 참고 컬럼입니다.)
⑥ 실물로 맞추기 (디렉터 전용)
🔒 실물로 맞추기 → 조정 대상 미리보기(전/후/증감) → 확인 시 부족·초과 품목만 실물 수량으로 조정하고 재고이동 기록을 남깁니다. 미실사·미등록은 절대 건드리지 않습니다(놓친 랙이 재고를 0으로 만들지 않도록).
🎁 프로모션 품목
장갑·티셔츠·모자 등은 프로모션 품목 관리에서 먼저 등록해야 실사에서 잡힙니다. 자동차부품 마스터와 분리되어 견적·매출엔 영향이 없습니다.
